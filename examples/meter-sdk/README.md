# Meter SDK examples

- `basic-usage.ts`: protect arbitrary work with prepaid credits.
- `mcp-tool.ts`: register an MCP tool with dynamic billing attribution.
- `nextjs-billing-route.ts`: redirect an authenticated buyer into the hosted portal.
- `webhook-route.ts`: verify a signed provider webhook using the raw body.
- `ai-cost.ts`: attach AI tokens and versioned model pricing to a committed transaction.

Install the relevant public packages and provide Meter credentials through
server-side environment variables before adapting these snippets.
